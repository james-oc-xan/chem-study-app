# GF Study App

Expo + TypeScript starter app with a basic camera screen powered by `expo-camera`.

## What’s included

- Fresh Expo SDK 54 TypeScript scaffold
- Camera permission request flow
- Live camera preview
- Front/back camera toggle button
- Camera permission plugin configuration in `app.json`

## Run locally

```bash
npm install
npm run start
```

Then press:
- `i` for iOS simulator
- `a` for Android emulator/device
- `w` for web (camera access depends on browser permissions)

## Notes

- iOS camera usage text is configured via the `expo-camera` plugin in `app.json`.
- This is a minimal foundation screen for expanding chemistry-study camera workflows next.
